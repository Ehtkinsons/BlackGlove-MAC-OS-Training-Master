Section 2 Review

Practice

  1. Complete iOS Setup Assistant on your iPad, iPod touch, or iPhone.
  2. Enable Location Services.
  3. Modify the following settings on your iOS device:
    1. Turn off Bluetooth
    2. Connect to a Wi-Fi network and turn off Ask to Join Networks
    3. Turn off Auto-Brightness
    4. Change Auto-Lock to 5 minutes
    5. Change the Lock Screen and Home Screen wallpaper
    6. Set a passcode
    7. Sign in with an Apple ID
  4. Download the Pages app
  5. Delete the Pages app from your device

End Goals

  • Complete iOS Setup Assistant
  • Navigate the iOS interface
  • Locate iOS device Information
  • Configure settings
  • Install and remove apps

Section 3 Review

Practice

  1. Set up a new local user account with the following information:
    1. Account Type: Administrator
    2. Full Name: Test User
    3. Account Name: testUSer
    4. Password: jamf1234
  2. Log into the testUser local user and set the following preferences:
    1. Change the Appearance color to Graphite
    2. Move the Dock to the left side and turn on Magnification
    3. Turn on Tap-to-click
    4. Turn on Printer Sharing
  3. Download and install MacTracker from the Mac App Store
  4. Using the Terminal app:
    1. Create a folder on the Desktop called "Review"
    2. Create a text file in the Review folder called "Review.txt"
    3. Create and executable script that:
      1. Includes the author, date created, and version number
      2. Includes a line to open Safari to the Jamf website

End Goals

  • Complete the macOS Setup Assistant
  • Use Finder, the Dock, Spotlight, and QUick Look to navigate macOS
  • Set user-specific and system-wide settings with System Preferences
  • Navigate macOS using the command-line interface with Terminal
  • Create a basic scripts to perform a sequence of commands
